# 阿瓦隆名詞介紹 - 湖中女神/湖上夫人
湖中女神是亞瑟王傳說裡的人物，傳說亞瑟王手中的王者之劍，即是由湖中女神賜予的。湖中女神又被稱為湖夫人、湖中仙子、湖的貴夫人、湖中妖女。

阿瓦隆遊戲規則書中，有一張「湖上夫人」的擴充卡，但通常大部分玩家會用湖中女神來稱呼此卡，或者是簡稱湖中、湖女。

規則書建議 7 人以上可選擇將湖中女神加入到遊戲中。
根據阿瓦隆百科多場遊戲的測試，建議**8 人以上**且有**莫德雷德**時再加入湖中女神，紅藍雙方的實力會較為平衡。

# 湖中女神擴充卡介紹
* 湖中女神的功能是可以讓持有的玩家查看場上一名玩家的身份，並選擇性說出該玩家的身份，而被查看身份的玩家會成為湖中的持有者。
* 遊戲一開始時湖中會被放置在最後一家 (起始玩家的上家)，此玩家被稱為首湖
* 湖中在第2、3、4局任務結束後使用，第2局湖中持有者為二湖，依此類推有三湖、四湖
# 湖中女神的使用方法
* 在第2、3、4局任務結束後，湖中的持有者會將『湖中女神卡』與『藍紅陣營卡』 傳給場上其中一位玩家(被湖者)
* 被湖者需要將代表自己『真實身份』 的陣營卡回傳給湖中持有者 (簡言之，好人傳藍卡，壞人傳紅卡)
    * <font color=#0000FF>藍卡：金色徽章中間有龍紋，代表正義方，也就是好人</font>
    * <font color=#FF0000>紅卡：銀色徽章中間有鳳紋，代表邪惡方，也就是壞人</font>
* 湖中持有者秘密查看陣營卡後，可保持沈默什麼都不說，或宣稱看到的結果
    * 湖中持有者沒有被規定必須說實話，但湖中持有者一定會知道被湖者真實的身份 (被湖者是壞人若回傳藍卡是違反遊戲規則的！)
    * 好人會希望場上的資訊都是正確的，因此湖中持有者若是好人基本上會都說實話說出被湖者的陣營
    * 除了湖中持有者、被湖者之外，陣營卡不可以給其他人看到，因此場上玩家必須自行判對湖中持有者說話的真實性
* 被湖者會成為下一位湖中持有者，且曾持有過湖中的玩家不可作爲被湖者
# 湖中女神相關名詞
* 好湖好：使用湖中與被湖的玩家都是好人
* 壞湖壞：使用湖中與被湖的玩家都是壞人
* 壞湖好：使用湖中的玩家是壞人
* 壞湖問號/X/?：使用湖中的玩家是壞人，被湖的玩家陣營未知
* 導湖：引導湖中去湖某位玩家
# 湖中女神使用範例
* 0 湖 8 說好 (看到藍卡), 8 湖 4 說壞 (看到紅卡)
* 0 代表起始湖 = 首湖
* 8 代表二湖
* 0 湖 8 說好：
    * 0 家對 8 家使用湖中
    * 8 家回傳真實陣營卡給 0 家
    * 0 家看完真實陣營卡後，宣稱自己看到的是好人
* 8 湖 4 說壞：
    * 8 家對 4 家使用湖中
    * 4 家回傳真實陣營卡給 8 家
    * 8 家看完真實陣營卡後，宣稱自己看到的是壞人
