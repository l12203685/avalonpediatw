# avalonpediatw UI 需求（Edward 2026-04-02）

## 設計原則
- **Mobile-first**：以手機垂直 layout 為主，電腦版配合手機，不做獨立桌面版
- 參考站：https://avalon.signage-cloud.org/

## Bug 修復（優先）
1. 任務結果/投票通過後卡住 — modal 沒自動關閉或 game state 沒推進
2. 第二局沒有湖中女神 — 規則：第二輪任務結束後持有者必須使用能力
3. Wiki 手機版 — sidebar 佔滿螢幕，文章被擠到最下面。改成收合或 top tabs
4. 座位手機版 — 全部擠在一起，需要 responsive

## 新功能：遊戲中牌譜紀錄頁面
- 用 grid 表格呈現即時牌譜
- 欄位：隊長 | 座位 0-9 投票狀態 | 結果 | 文字紀錄
- 任務結果行：黃色背景，顯示 o/x（成功/失敗）
- 湖中女神行：藍色背景，顯示 A>B o/x
- 手機垂直滾動
- 對應牌譜速記格式（如：`170` = 座位 1 當隊長派 7,0；`ooo` = 全成功）

- 不要圖像，純文字+圖標
