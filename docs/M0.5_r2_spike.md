## 2. R2 cost estimate

Assumptions: Cloudflare R2 pricing (public rate card as of 2025).

| Line | Value |
|---|---|
| Total storage | ~0.0484 GB (49.56 MB) |
| Storage rate | $0.015 / GB-month |
| Raw storage cost | ~$0.000726 / month |
| Free tier | 10 GB storage + 1M Class-A ops + 10M Class-B ops / month |
| **Effective cost** | **$0.00 / month** (entirely inside free tier) |
| Egress | $0 (R2 has zero egress fees — main reason to pick R2 over S3) |

Even at 100× growth (~5 GB), still free. R2 only starts billing past 10 GB stored.

## 6. Upload script skeleton (NOT RUN)

File: `scripts/upload_videos_r2.py` (not yet created — skeleton only).

```python
#!/usr/bin/env python3
"""Upload Avalon short videos to Cloudflare R2.

Reads credentials from ~/.claude/credentials/.env (R2_ACCOUNT_ID, R2_ACCESS_KEY_ID,
R2_SECRET_ACCESS_KEY, R2_BUCKET). Never hard-code.

Usage:
    python scripts/upload_videos_r2.py --dry-run
    python scripts/upload_videos_r2.py --bucket avalonpediatw-media
"""
from __future__ import annotations
import argparse
import os
import sys
from pathlib import Path

import boto3
import yaml
from botocore.config import Config

SRC_DIR = Path(r"C:/Users/admin/GoogleDrive/專案/阿瓦隆百科/assets/videos")
MANIFEST = Path(__file__).resolve().parents[1] / "content/_data/video_manifest.yaml"
KEY_PREFIX = "videos/shorts/"


def r2_client(account_id: str, access_key: str, secret_key: str):
    endpoint = f"https://{account_id}.r2.cloudflarestorage.com"
    return boto3.client(
        "s3",
        endpoint_url=endpoint,
        aws_access_key_id=access_key,
        aws_secret_access_key=secret_key,
        config=Config(signature_version="s3v4", region_name="auto"),
    )


def main() -> int:
    ap = argparse.ArgumentParser()
    ap.add_argument("--bucket", default=os.environ.get("R2_BUCKET", "avalonpediatw-media"))
    ap.add_argument("--dry-run", action="store_true")
    args = ap.parse_args()

    manifest = yaml.safe_load(MANIFEST.read_text(encoding="utf-8"))
    videos = manifest["videos"]

    if args.dry_run:
        for v in videos:
            src = SRC_DIR / v["original_filename"]
            key = f"{KEY_PREFIX}{v['slug']}.mp4"
            size_mb = src.stat().st_size / 1024 / 1024 if src.exists() else -1
            print(f"[dry] PUT s3://{args.bucket}/{key}  <- {src.name} ({size_mb:.2f} MB)")
        return 0

    acct = os.environ["R2_ACCOUNT_ID"]
    ak = os.environ["R2_ACCESS_KEY_ID"]
    sk = os.environ["R2_SECRET_ACCESS_KEY"]
    client = r2_client(acct, ak, sk)

    for v in videos:
        src = SRC_DIR / v["original_filename"]
        key = f"{KEY_PREFIX}{v['slug']}.mp4"
        print(f"PUT s3://{args.bucket}/{key}  ({src.stat().st_size:,} bytes)")
        client.upload_file(
            Filename=str(src),
            Bucket=args.bucket,
            Key=key,
            ExtraArgs={
                "ContentType": "video/mp4",
                "CacheControl": "public, max-age=31536000, immutable",
                "Metadata": {
                    "title-zh": v["title_zh"],
                    "role-tag": v["role_tag"],
                    "category": v["category"],
                },
            },
        )
    return 0


if __name__ == "__main__":
    sys.exit(main())
```

Alternative tools (also viable, not chosen yet):
- `rclone` with an `[r2]` remote — good for one-shot sync, no Python deps.
- `wrangler r2 object put` — official CF CLI, single-file oriented, no bulk loop.

Recommendation: boto3 script above because it reads the manifest, sets metadata + cache headers in one pass, and is reusable for future batches.

## 7. Public access pattern

Two options — decision deferred.

| Option | URL shape | Pros | Cons |
|---|---|---|---|
| **A. R2 public bucket** (enable "Public access" on bucket) | `https://pub-<hash>.r2.dev/videos/shorts/assassin-key.mp4` | Zero extra config | Ugly URL, no CDN cache rules |
| **B. Custom domain via Workers** | `https://media.avalonpedia.tw/videos/shorts/assassin-key.mp4` | Branded, CDN-cached, revocable | Needs domain + Worker route |

Recommend **B** long-term, **A** acceptable for M0.5 spike.
