# Cloudflare R2 Setup — avalonpediatw-media

> Owner: Edward
> Last updated: 2026-04-14 09:59 +08
> Goal: 建立 public R2 bucket 存放角色影片 + 未來上線大體積媒體，取代 GitHub repo 承擔媒體。

## Why R2

- S3 相容 API（boto3 直接用）
- **Zero egress fee** — 比 S3/GCS 便宜很多，適合公開 media CDN
- 10 GB/month free tier
- 可綁 custom domain + Cloudflare CDN

---

## Step-by-step

### 1. 開 Cloudflare 帳號

- https://dash.cloudflare.com/sign-up
- 用 Edward 的主 email（與 GitHub 同 or 獨立都可）
- 啟用 2FA（TOTP / passkey）

### 2. 建 R2 Bucket

1. Dashboard → **R2 Object Storage** → **Create bucket**
2. Name: `avalonpediatw-media`
3. Location: **Automatic**（Cloudflare 自動選最近 region）
4. Default storage class: **Standard**
5. Create

### 3. 啟用公開存取

有兩個選項，挑一個：

**A. r2.dev 臨時公開網址（最快）**
1. Bucket → **Settings** → **Public access** → **Allow Access**
2. 拿到 `https://pub-<hash>.r2.dev/` base URL
3. 設為 env `R2_PUBLIC_BASE`

**B. Custom domain（推薦，生產用）**
1. Bucket → **Settings** → **Custom Domains** → **Connect Domain**
2. Domain: `media.avalonpediatw.com`
3. Cloudflare 自動幫你建 CNAME（前提是 `avalonpediatw.com` 已在 Cloudflare DNS）
4. 等 SSL cert 發完（~1 min）
5. 設 `R2_PUBLIC_BASE=https://media.avalonpediatw.com`

### 4. 產 API Token

1. Dashboard → **R2** → **Manage R2 API Tokens** → **Create API Token**
2. Permission: **Object Read & Write**
3. Specify bucket: `avalonpediatw-media`（最小權限）
4. TTL: 1 year（可續）
5. Create → **立刻抄下** Access Key ID + Secret Access Key（關頁就看不到）
6. 順便記 Account ID（在 R2 首頁右欄）

### 5. 寫入本地 `.env.r2`（不進 git）

在 repo 根目錄建 `.env.r2`（已 gitignore）：

```bash
R2_ACCOUNT_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
R2_ACCESS_KEY_ID=xxxxxxxxxxxxxxxxxxxx
R2_SECRET_ACCESS_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
R2_BUCKET=avalonpediatw-media
R2_PUBLIC_BASE=https://media.avalonpediatw.com
```

載入：`set -a; source .env.r2; set +a`（bash）或 `Get-Content .env.r2 | ...`（PowerShell）。

### 6. 執行上傳

```bash
cd C:/Users/admin/workspace/avalonpediatw
pip install boto3
python scripts/upload_r2.py --dry-run    # 驗證清單 + source path
python scripts/upload_r2.py              # 真正上傳 + 回寫 EXTERNAL_REFS.md
```

上傳後：
- `assets/EXTERNAL_REFS.md` 的 `R2 URL (pending)` 會變成實際 URL
- `git add assets/EXTERNAL_REFS.md && git commit -m "chore(assets): R2 URLs for character videos"`
- PR 進 main

### 7. 驗證

```bash
curl -I https://media.avalonpediatw.com/characters/merlin.mp4
# 預期 200 + content-type: video/mp4 + CF cache headers
```

## Rollback / 刪除

- 刪 bucket：R2 → bucket → Settings → Delete bucket（**會清所有 object**）
- 只撤 token：Manage R2 API Tokens → Roll / Delete
- 解除 custom domain：Settings → Custom Domains → Disconnect

## Cost 估算

| 項目 | 預估 |
|------|------|
| 儲存 | 52 MB → free tier 內 |
| Class A ops（put） | 6 次 / month → free tier 內 |
| Class B ops（get） | ~ 幾千次 / month → free tier 內 |
| Egress | $0（R2 不收） |

**預估月費 = $0**（在 free tier 10 GB 內）。

## Security notes

- R2 API token 只給 `avalonpediatw-media` bucket 寫權（最小權限）
- `.env.r2` 必須在 `.gitignore`
- Public access 只放**確定可公開**的素材；Discord 圖庫**絕不上傳**
- 若 token 洩漏：Manage R2 API Tokens → Roll 立刻換
