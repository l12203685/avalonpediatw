# 派票之前

## 思考場上的資訊流
* 前面的人派了什麼
    * 解讀場上的資訊流 (客觀/旁觀視角)
    * 是否有獨立資訊/雙重假設/邏輯一致性
    * 是否有換位思考/跟隨場上共識/平衡局勢
* 代入自己的身分與資訊
    * 加入自己的身分與資訊後解讀場上的資訊流 (主觀視角)
    * 選擇自己的組合並回頭檢查合理性/是否突兀/難以解讀/容易造成誤解
    
## 帶有意識
* 不存在純隨機派票
    * 除了是 1-1 的忠臣, 每次派票都是有意識/意義的
    * 後面的派票會延續前面的派票展開 (這是指邏輯性地展開)
    * 以解題比喻，紅方負責布局出題，藍方負責解題破局
* 什麼叫做帶有意識
    * 清楚瞭解自己任務選擇背後的前提
    * 清楚瞭解自己任務選擇要選哪些人
    * 清楚瞭解自己為什麼做這樣的選擇
    * 清楚瞭解自己任務選擇是表達什麼
    * 清楚瞭解自己這樣選擇後希望達到的效果與目的

# 派票之後

## 思考場上的資訊流
* 後面的人派了什麼
    * 在你後面的玩家是否有根據你的派票作出反應
    * 資訊交叉比對下，場上是否認同你的資訊
* 代入自己的身分與資訊
    * 後面的人是否對你的身分/陣營有誤解
    * 後面的人是否"利用"你的資訊做誤導

## 即時修正
* 每個動作後都要思考
    * 你的效果跟目的是否有達到
    * 資訊是否正確傳遞出去/任務引導是否成功/讓場上認同你是好人
    * 如果局勢不如預期，即時修正並想辦法走向你期望的結果
* 隨機應變
    * 派票到投黑白球並非連續動作
    * 隨時因應場上的派票/黑白球去做應變
    * 也可以隨機變硬
