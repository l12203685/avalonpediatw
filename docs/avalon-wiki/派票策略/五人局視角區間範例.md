# 進過任務越多次的人, 視角與區間會越清楚
* 假設第一局 1-5 由 {3, 5} 出任務, 一張任務失敗
    * 此時場上的共同區間為
        * {3, 5} 至少一壞
        * {1, 2, 4} 最多一壞
    * 隊內玩家的好人視角區間為
        * 3: {5} 一壞, {1, 2, 4} 一壞
        * 5: {3} 一壞, {1, 2, 4} 一壞
    * 隊外玩家的好人視角區間為
        * 1: {2, 4} 最多一壞, {3, 5} 至少一壞
        * 2: {1, 4} 最多一壞, {3, 5} 至少一壞
        * 4: {1, 2} 最多一壞, {3, 5} 至少一壞
* 假設第二局 2-5 由 {2, 4, 5} 出任務, 一張任務失敗
    * 此時場上的共同區間為
        * {3, 5} 至少一壞
        * {1, 2, 4} 最多一壞
        * {2, 4, 5} 至少一壞
        * {1, 3} 最多一壞
    * 1 家好人視角區間為
        * 若 {3/5} 為好人
            * {5/3} 為壞
            * {2, 4} 為一壞區間
        * 若 {3, 5} 皆為壞人
            * {2, 4} 為兩好
        * 因此對於 1 家最保險的任務組合應為 {1, 2} 或 {1, 4}
    * 2/4 家好人視角區間為
        * 若 {3/5} 為好人
            * {5/3} 為壞
            * {1, 4/2} 為一壞區間
        * 若 {3, 5} 皆為壞人
            * {1, 4/2} 為三好
    * 3 家好人視角區間為
        * {5} 為壞人
        * 若 1 為壞人 {2, 4} 為兩好
        * 若 1 為好人 {2, 4} 為一壞區間
    * 5 家好人視角區間為
        * {3} 為壞人 {2, 4} 為一壞區間
        * {1} 必為好人 (`鐵好`)
        * <font color=#FF0000>**僅有 5 家好人時, 有 {1} 鐵好的視角**</font>
# 加入能力者絕對視角後, 不可能的區間會被移除
* 假設角色對應如下: 
```json
{1: 梅林, 2: 派西, 3: 忠臣, 4: 刺客, 5: 莫甘}
* 派西視角 {1, 5} 必出一壞, 因此派西可知 1 好, 5 壞
* 更直觀的派西視角是: 5 若為梅林, {2, 4, 5} 不會任務失敗
```
* 假設角色對應如下: 
```json
{1: 忠臣, 2: 莫甘, 3: 派西, 4: 梅林, 5: 刺客}
* 派西視角 {2, 4} 必出一壞, {5} 鐵壞
* 因此派西視角可在第一局結束後馬上知道 1 鐵好
```
* 假設角色對應如下:
```json
{1: 梅林, 2: 刺客, 3: 莫甘, 4: 忠臣, 5: 派西}
* 派西視角在第一局結束後可知 1 鐵好
* 派西視角在第二局派出必有一壞的 {2, 4, 5} 組合, 目的可能是為了不出賣梅林
```
* 假設角色對應如下: 
```json
{1: 派西, 2: 莫甘, 3: 刺客, 4: 梅林, 5: 忠臣}
* 派西視角中, {2, 4} 一壞, {3, 5} 一壞
* 這種組合是最難消除疑慮區間的, 因為兩局任務都是派西已知的一壞區間全出任務
* 類似的組合還有 {1: 派西, 2: 梅林, 3: 忠臣, 4: 莫甘, 5: 刺客}
```
