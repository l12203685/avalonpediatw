# 壞人怎麼被抓到的
* 當你已經學會看簡單的區間，你可能會注意到拿到壞人牌，尤其是拇指牌時，自己經常處於"場上共同視角裡有壞的區間"內
* 假設你想順著當下該場上訊息走（即你認同"場上自己所在的區間有壞"的這個訊息），這時候切記忌諱去派與你共同區間內的其他牌
* 這相當於在告訴其他人，＂哦，對方比我好，我才是那個壞人！＂

## 派票範例
* 9 家奧伯倫
    ```
    168 7+
    148 2+
    345
    267 4+
    568 
    oox

    2670
    2478
    2780
    ```
    * 你是9家奧伯倫，這時候你2-4派了2790
    * 等同是默認了前面124678多好、35兩壞、90可能要至少有一隻的資訊
    * 且不說壞人數目的問題，這時候你又派了0，那你又是什麼？
    * 0肯定是會派自己，肯定不會優先考慮到你，這就是一種無意義的買票行為。


* 另一個類似概念的八人局牌局範例：
    ```
    147
    267
    135
    146
    156 
    oox
    ```
    * 在此一範例中，對5家而言比較友善的是147
    * 可能是說23一隻，56一隻
    * 此時選擇6的2家與4家對5來說可能就不那麼友好
    * 5卻選1又選6，那對其他人而言自然也是選6也不會去選5
