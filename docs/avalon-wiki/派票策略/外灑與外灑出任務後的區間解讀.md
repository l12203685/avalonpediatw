# 動機或原因
* 嘗試改變尾派
* 增加未來走勢分支, 提供空間 見[遊戲分支 ](/uaUE9FZ9T-y_NMFVwURUyQ)
* (藍方) 傳遞更多資訊與視角 
* (紅方) 傳遞錯誤資訊與擾亂視角
* (紅方) 送隊友出任務 

# 確認區間/讓區間顯露
## 假設 1-1 ~ 1-4 的某位玩家 H 外灑, 場上剛好 6 顆白球
* 場外兩白玩家為 O1, O2, 場內玩家為 I1, I2, I3, 四顆黑球為 B1, B2, B3, B4
* 任務結果為一張失敗, 此時場上的共同區間如下
* {I1, I2, I3} 至少一壞, {H, O1, O2} 為一個區間, {B1, B2, B3, B4} 為剩下的區間
* 白話就是常聽到的: 場內至少一壞, 場外白區間, 場外黑球區間
## 有區間之後的派票選擇
* 第二局在選擇時, 通常就不會考慮同時選同一區間內的玩家 
    * 例如: O1 不太可能派 [O1, O2, H, I1] 這個組合
    * 場外白的玩家不太可能再同時選擇外灑的玩家與外白的玩家
* 原因是, 除了第一局任務外, 剩下兩個區間通常不太可能是"完全同陣營"的玩家
    * 反過來說, 一旦選擇了同一區間內所有玩家, 傳遞出的資訊就是"認為某個區間有兩壞以上"
    * 例如: B1 派 [B1, H, O1, O2] 這個組合
    * 表示某玩家選擇了所有場外白的組合, 並且出任務的玩家全部不選
    * 表示 B1 視角中 {I1, I2, I3} 或 {B2, B3, B4} 其中一個區間有兩個以上的壞人
    * 表示 B1 視角中"任務內有兩壞以上"或"跟他一起開外白的有兩壞以上"
* 注意 [H, O1, O2] 這個區間的玩家視角會特別重要
    * 除了玩家數少之外, 他們還是讓第一局任務失敗的三顆場外白球
* 最常見的幾種區間思考方式 (回顧[鐵邏輯/正邏輯/反邏輯](/qVezcBzvTWOd1xTzBIr51Q))
    * 思考的起點: 
        * 外灑的人為好人/壞人 
        * => 對跟同區間另外兩個場外白是偏好/偏壞 
        * => 剩下的場外黑區間的壞人人數定義
    * H 為好人, O1, O2 有壞人隊友幫忙衝, {B1, B2, B3, B4} 有兩個壞人
    * H 為壞人, O1, O2 有壞人隊友幫忙衝, {B1, B2, B3, B4} 只有一個壞人
    * H 為壞人, O1, O2 是兩個幫衝的好人, {B1, B2, B3, B4} 有兩個壞人
    * H 為好人, O1, O2 是兩個幫衝的好人, {B1, B2, B3, B4} 有至少兩個壞人
* 結合個人視角與場上資訊
    * 自身屬於哪個區間 & 派票的邏輯
    * 開外白代表可能反對當下的雙尾派 (雙尾派至少一壞)
    * 就可以找出相對合理的區間是上述何者
* 如果想假設極端狀況
    * 場內為三壞, 三個外白為三個忠臣, 四顆黑球只有一位忠臣, 剩下皆為能力者
    * 假設某位外白的玩家是這樣的視角, 那他就會派出三個外白 + 外黑區間的某位玩家 ([H, O1, O2] + B)
    * 但這種假設下, 依然可以選擇場外白只選一, 場外黑選二
        * 這個選法容錯率更高
        * 就算外白內黑區間是錯誤假設, 只要第一局多壞表達就是正確的
    * 表示後者的選法優於前者
        * 不同的前提假設, 都有相同的符合派票選擇
        * 沒有理由選擇一個劣勢策略/廣泛性低的策略
        * 詳細參閱[優勢策略](/w7XVeZRHSQqsLpvp-hsVMw)與[廣泛性](/uaUE9FZ9T-y_NMFVwURUyQ)

# 牌局範例

## 視角與區間
* 旁觀視角區間
    * 236 至少一壞, 190 高機率一壞, 4578 一至二壞
* 3 家好人主觀視角區間
    * 26 至少一壞, 190 高機率一壞, 4578 一至二壞
* 輪到 3 家派票, 此時思考方向
    * 26 是至少一壞, 因此應該都不要選擇
    * 190 高機率一壞, 其中合理程度應該是 1 > 0 > 9, 理由如下
        * 1 是先派出 236 組合的人, 因此同樣組合開白很合理
        * 0 如果是壞人, 沒必要開白 236 的隊友, 導致少掉自己幾乎可以拿下任務的機會
        * 9 的白比較沒有邏輯, 沒有足夠的證據支持開這顆外白, 偏嘗試性白球
    * 4578 一至兩壞
        * 7 如果是相認壞: 190 的隊友幫忙衝刺隊友同時搶尾派, 想快速拿下三紅 
        * 5 考慮到尾派被洗掉, 可能是裡面偏好的玩家
        * 4 & 8 則是幾乎無關, 但通常此時最後一壞會在這個區間內 (俗稱被擠出來的壞人)
    * 再配合外白的位置, 解讀玩家之間的立場
        * 0 寧願放棄自己的第二局尾派也不願意讓 5 有第一局尾派
        * 因此 5 & 0 很像至少一壞且 150 不像全好
* 綜合以上, 可以從 50 有幾個壞人開始定出合理的派票
    * 如果 50 一壞
        * 比較合理的組合就是 1/9/0 挑一個, 4578 挑兩個
            * 但因為 5/0 一壞, 而且 150 不像全好, 選擇上會盡量排除 5/0 同進的狀況
            * 通常以遊戲分支角度, 選擇 1 是較為合理的選項
            * 但也有些人會認為 1 是派隊友洗隊友 (12560 至少三壞) 
        * 假設選 0, 4578 就會只剩 478 可以選擇
        * 假設選 5, 478 就只會挑一個人, 剩下的再從 1/9 去做選擇
        * 假設選 1, 5/0 會被直接排除, 剩下的組合就只剩 478 選二
        * 假設認為 150 兩壞/ 26 至少一壞的情況: 7幾乎是好人, 這時合理的組合應該是 379 + 4/8 選一
    * 如果 50 兩壞 (隊友洗隊友): 2560 三壞
        * 比較合理的組合就是 1/9 挑一個, 478 挑兩個
        * 如上所述, 1/9 區間比較高機率會選擇 1, 且在 50 兩壞的前提下, 1 做的是好人行為
        * 如上所述, 478 區間認為誰比較壞, 通常會認為最後一壞在 4/8 之中
        * 因此最後較合理的選擇是 137 + 4/8 選
        * 另外有些人會認為 7 有尾派, 因此直接排除不選7, 這也是合理的想法
        * 從 1 偏好篇壞出發
        * 1 偏好, 5/0至少一壞
        * 1 偏壞, 5/0最多一壞
        * 加上 0 自己的視角
* 假設 3 是梅林, 拇指在 450
    * 上述討論中符合這個情形的有 
    * 150 兩壞/ 26 至少一壞: 379 + 4/8 => 3789
    * 50 兩壞: 2560 三壞 => 137 + 4/8 => 1378
    * 可以發現幾乎所有組合都是 378 當底, 1/9 選
