# 能力者
* 十人局中能力者有六位其中有四位皆是壞人
* 當你發現某人是能力者的時候高機率是壞人
* 當你被察覺到你是能力者的時候
* 因為不確定你是好人還是壞人能力者時，派票很容易避開你
* 如果你是壞人能力者雜魚會避開你、好人能力者不會派你
* 如果你是好人能力者雜魚會避開你、壞人能力者會發現你 (非派即梅) 

# 能力派票
* 要怎麼發現能力者，大多數是察覺到該家「能力派」
* 什麼是能力派
    * 當下給場上目前跟不上的資訊 
        * Ex: 壞人為了博取好人信任所以派全好組合, 但這個全好不符合邏輯那本身就是能力派的
    * 刻意本身就是能力的展現
    * 擁有獨立資訊的玩家
* 牌局範例
```json
169
469 2+
358
480
250 oxx
```
* 旁觀視角解析 
    * 169: 待後續派票定義好壞
    * 469: 2家覺得50有壞，嘗試外撒洗尾派
    * 358: 忽視2家資訊，直接派5家(能力者，且能區分5比0好)
    * 480: 忽視2家資訊，未定義169；忽視3家資訊(認為5好人)，表達0比5好，且高機率表示59雙壞*
*註1:有些人會覺得在1-1~1-4的時候 50雙尾家突然派其中一家，是表示沒派到另外一位偏壞，另外如果9是好人，不會直接派8去跳過9這就是我們俗稱的夾人
* 配置:刺娜德奧派梅 291534
* 牌局結果:三藍但壞人皆刺梅林
