# 前言 By 皓子
給對遊戲還沒有概念或是玩過但總感覺那裏怪怪的朋友，這是綜合許多人的經驗分享。

# 壞人篇
* 壞人不是玩躲貓貓，不是藏到最後就是贏家
* 不敢出失敗的壞人沒資格當壞人
* 不要怕被發現是壞人，要怕的是連失敗都還沒出到就被發現是壞人
* 兩張失敗不是世界末日
* 壞人是用最少資訊的代價換取最多的紅燈
* 只會喊炸車的壞人，只是一個沒夢想的鹹魚尤其是1-5
* 第一局開紅，第二局3+1炸車，是因為可以封鎖資訊且能毀湖中
* 還沒被發現是壞人時，請不要自己跟全世界說
* 被發現是鐵壞時拿出你最後的價值吧，沒錯就是那顆白球
* 沒人有正確的資訊時不用刻意賣隊友，沒人會感謝你

# 梅林篇
* 玩梅林盡力就好，存活靠雜魚
* 梅林最重要的任務就是釋放資訊引導好人派對 
* 學著去釋放資訊的同時隱藏自己是擁有資訊的人
* 第一局開藍燈，裡面有藏不用緊張，輪次終究還領先
* 拇指450那個叫絕望
* 梅林盡量不要投降，就算大家都派錯了也請讓其他人練習派票
* 梅林若非必要別輕易踩拇指，相信隊友
* 自己派的拇指還是要引導出來

# 派西篇
* 派西不要盲踩拇指，那不叫試試看，那叫賭博
* 某方面來說跳派是剝奪其他玩家的體驗請盡量不要這樣做
* 你有跳派的權利，我也有不信你的理由
* 沒看派票之前建議雙下，看完派票之後嘗試站邊
* 認錯沒關係，三紅之前都來得及修正
* 釋放資訊同時隱藏是誰給你資訊，相信隊友
* 十個跳派九個爛

# 觀念篇
## 湖中女神
* 湖中女神是找好人，不是要湖誰證明誰是壞人
* 湖中說壞不代表另一個就一定是好人
## 派票/任務選擇
* 派票是假設，出任務是驗證，最後才是修正，
* 派票的時候請假設你的四壞在那裏，若假設不出來代表你一定派錯，玩好人如此玩壞人更是 
* 寧願派到三壞也不要把莫甘娜和梅林派在一起
## 狀態流?
* 說話是一門藝術，派票是一種技術，最好的遊戲環境是30%靠說話70% 靠派票
* 貼臉、起狀態是最不入流的玩法
* 人可以用演的用說的用騙的，但任務結果不會騙你，你湖的人也不會騙你
* 派票就是說話，你可以不說話但不能不派票
## 好人相關
* 拿到藍燈是最重要的任務
* 十個男人九個壞，能力者也一樣
## 能力者相關 
* 你怎麼玩雜魚你當能力者就怎麼玩
* 讓別人發現你是能力者是輸的開始
* 比起讓人知道你是能力者，先讓別人相信你是好人比較重要
* 能力是比較出來的  
## 黑白球相關
* 收起你那個蠢蠢欲動的手，不要隨便外白內黑，尤其你是尾家
* 你可以外白內黑但你要知道這動作的意義和效益
## 其他
* ==阿瓦隆潛規則只有一條：兩紅後壞人必勝請馬上結束==
    * 尾派壞人炸一炸
    * 尾派和前一位是相認壞，炸一炸
    * 尾派前一位隊伍中某人和尾派都是拇指（梅林出來講）
    * 注意：若有保護局，請確定必勝再出來喊
* 老手跟新手最大的差別，是邏輯 
* 有我就白，沒我就黑，第一局開藍燈永遠3+1好聽點叫不犯錯，難聽點叫躺分
* 嘗試不一定會贏，但不嘗試一定不會進步
* 要進步先復盤
* 老手總愛說看情況，不是敷衍是真的要看情況
* 兩個最少出一壞跟兩個人最多出一壞差別很大
* 在確定三壞的情況下湖中可以用來確認第四壞

# 自我審視進化論 
* 想想你是那一層呢
* 入門階段:對遊戲目的是什麼根本不知道，就算玩壞人也不敢出失敗怕被發現是壞人或怕兩張失敗
* 新手階段:對遊戲有基本認知，壞人會出失敗，但玩好人不知道怎麼找其他好人，看不懂派票資訊
* 中手階段:會知道怎麼找資訊了，但玩梅林容易送頭或是玩雜魚過度被動等場上風向給資訊
* 老手階段:開始會推理，壞人開始會玩戰術利用正反邏輯去刪掉雜魚 梅林也開始知道不用亮著打
* 神仙階段:好人很會吸收資訊，修正區間也很快 任務容易三藍，但壞人也會去找資訊源頭去刀梅林，壞人配合下有機會拿下三紅
