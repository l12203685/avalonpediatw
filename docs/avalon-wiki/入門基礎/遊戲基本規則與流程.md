# 遊戲說明
* 阿瓦隆為一種陣營遊戲, 分別為藍方好人陣營與紅方壞人陣營
* 藍方具有人數優勢; 紅方具有資訊優勢
* 兩個陣營有各自需要達成的目標, 不論何時都<font color=#FF0000>**請清楚知道自己所屬陣營的遊戲目標!!**</font>

# 遊戲目標
## 藍方陣營目標: 完成以下**所有**條件
* 在每一局, 避免任務組合被反對超過四次 => 幾乎都會達成, 原因請看名詞說明`尾派`
* 任務結果成功三局 (簡稱: `三藍`)
* 避免梅林被刺客刺殺 (簡稱: `躲刺`, `躲刀`)
## 紅方陣營目標: 完成以下**其中一個**條件
* 讓任何一局任務組合被反對超過四次 => 幾乎無法達成, 原因請看名詞說明`尾派`
* 任務結果失敗三局 (簡稱: `三紅`)
* 引導刺客正確指認梅林並刺殺成功

# 角色說明
* 邪惡能看清自己，梅林卻關照未來

## 正義方
* 梅林: 知道誰是邪惡方，但必須隱藏好自己身份
    * 梅林看的到除了莫德雷德以外的所有邪惡方
    * 讓刺客發現你是梅林，只要被正確指認後依然算邪惡方勝利
* 亞瑟的忠臣
* 派西維爾: 知道誰是梅林
    * 知道梅林的位置
    * 莫甘娜會以梅林形象出現
    * 在夜晚階段派西會看到兩根姆指
    * 但無法區分莫甘娜跟梅林是誰

## 邪惡方
* 刺客: 莫德雷德的爪牙
    * 找到梅林，並在任務三藍後正確指認
* 莫德雷得的爪牙
    * 普通的壞蛋
* 莫甘娜: 以梅林的形象出現
    * 夜晚會被派西維爾知道你與真梅林的位置
    * 但無法區分莫甘娜跟梅林是誰
* 莫德雷德: 梅林不知道他
    * 唯一不會被梅林看到的壞人
* 奧伯倫: 對邪惡方來說彼此互不了解
    * 夜晚邪惡方相認時不能睜眼
    * 不會知道邪惡方位置

# 遊戲流程
1. 抽取角色卡, 並決定起始玩家 (`1家`)
    ![腳色卡](https://i.imgur.com/bwz2yAz.png)
    ![腳色卡](https://i.imgur.com/gGVIOMC.jpg)
    * 若有配製湖中女神 (簡稱`湖女`, `湖中`), 則將其交給起始玩家上一位玩家 (`0家`)
    * 湖女的使用時間點: 第二, 三, 四局任務的 **{任務階段}** 結束後使用 
2. 夜晚陣營確認
    * 奧伯倫以外的紅方豎起拇指, 睜眼互相確認拇指位置
    * 莫德雷德以外的紅方豎起拇指, 梅林睜眼確認拇指位置
    * 梅林與莫甘娜豎起拇指, 派西維爾睜眼確認兩根拇指的位置
3. 由起始玩家開始, 順時針進行以下流程
    * **{隊伍組合選擇階段}**
        1. 隊長將任務指示牌 (盾牌) 分派給任務組合人選
        2. 隊長宣告確定任務組合後, 進入 **{投票階段}**
    * **{投票階段}**
        1. 玩家對於任務組合選擇贊成(白球)或反對(黑球)後反面覆蓋
        2. 所有玩家確定自己的選擇後同時翻開, 並依據下述情況進行動作
            * 若贊成票總數未超過遊戲玩家一半
                1. 當局任務組合被反對累積次數增加 1 次
                2. 若當局任務組合被反對累積次數到達 5 次, 則紅方陣營獲勝
                3. 若當局任務組合被反對次數未達 5 次, 則順時針傳遞隊長標記
                4. 由下位玩家擔任隊長, 並回到 **{隊伍組合選擇階段}**
            * 若贊成票總數超過遊戲玩家一半, 進行 **{任務階段}**
    * **{任務階段}**
        1. 任務組合內的玩家會各領取一張**任務成功**和一張**任務失敗**卡
            ![任務成功與任務失敗](https://i.imgur.com/1MjKR2b.png)    
        2. 任務組合內的玩家選擇要出的任務卡, 並覆蓋後打出(不能讓任何一位其他玩家知道)
        3. 藍方陣營玩家僅能選擇**任務成功**; 紅方陣營玩家可選擇**任務失敗**或**任務成功**
        4. 將所有玩家打出的任務卡洗勻後一起打開並結算任務結果
            * 計算**任務失敗**總數
            * 若**任務失敗**卡張數 = 0 (全部皆為**任務成功**), 該局由藍方陣營拿下 (簡稱`開藍`)
            * 只要出現一張以上**任務失敗**, 該局即由紅方陣營拿下 (簡稱`開紅`)
            * 遊戲人數 >= 7 的第四局 (也稱作`保護局`) 的情況
                * **任務失敗**卡張數必須 >= 2, 才會 `開紅`, 否則都會 `開藍`
                    ![](https://i.imgur.com/YGnDRpM.png) 
                    ![](https://i.imgur.com/2astsQm.png) 
                * 稱為保護局的原因
                    * 保護藍方陣營幾乎能使第四局開藍
                    * 第四局開藍代表第四局的任務組合最多一壞, 或兩不相認壞
                    * 配合第四局任務組合與最後的湖中女神, 在決勝局前可獲得極大量資訊
        5. 標示任務結果 (藍 or 紅) 後, 判斷要進入哪個階段
            * 判斷雙方陣營第一目標條件是否達成: `三藍` or `三紅`
                * 若`三紅`條件達成, 則**紅方獲勝**
                * 若`三藍`條件達成, 則進入 **{刺殺階段}**
                * 若皆未達成, 判斷是否進入 **{湖中階段}**
            * **{湖中階段}** 結束後, 或不用進入 **{湖中階段}**, 或未配置湖女
                * 傳遞隊長標記, 下一局開始
                * 進入 **{隊伍組合選擇階段}**
    * **{湖中階段}**
        1. 湖上夫人(`湖女`)的使用時間點: 第二, 三, 四局任務的 **{任務階段}** 結束後使用
        2. 持有湖女的玩家, 將陣營卡與湖女交給被湖的玩家
        3. 被湖的玩家**必須將自己的真實陣營卡回傳**給使用湖女的玩家
        4. 用湖玩家**可以選擇**是否宣告結果(好/壞, 藍/紅, 正義/邪惡, 龍/鳳), **亦可以選擇說謊**
        5. 湖中階段結束
    * **{刺殺階段}**
        1. 根據規則書說明, 此時只有刺客可將底牌翻開
        2. 只有邪惡方(紅方)可以討論刺殺對象, 且正義方(藍方)在刺殺階段不可發言
        3. 紅方可透過發言得知彼此角色, 因此等同所有邪惡方 (包含奧伯倫) 一起翻牌討論
        4. 最終選擇權依然在刺客手中, 若指認對象為梅林則該局紅方獲勝, 反之則為藍方獲勝
            > Note:
            有些地方的 Home Rule 是奧伯倫不可參予討論刺殺
            這些 Home Rule 都是為了增減紅藍方相對強度或遊戲變化性
            並沒有絕對的好壞, 只要大家具有共識即可
            > 
4. 各階段簡略流程如下
    * **{隊伍組合選擇階段}** => **{投票階段}**
        * 白球未過半 => 移交隊長 => 新一輪 **{隊伍組合選擇階段}**
        * 白球過半 => **{任務階段}** => 任務結果 => 判斷是否達成遊戲目標 (`三藍`/`三紅`)
            * 皆未達成 => **{湖中階段}** => 新一輪 **{隊伍組合選擇階段}**
            * `三紅` => 紅方獲勝
            * `三藍` => **{刺殺階段}**
                * 刺客指認梅林正確, 刺殺成功 => 紅方獲勝
                * 刺殺指認梅林錯誤, 刺殺失敗 => 藍方獲勝
