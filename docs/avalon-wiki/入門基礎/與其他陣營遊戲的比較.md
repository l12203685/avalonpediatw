# 本文對陣營遊戲的定義
* 將玩家分成兩或多陣營 (排除一人一陣營的策略遊戲)
* 大部分玩家不會知道其他人真實身分 (排除山中小屋、機密代號等)
* 各陣營有獨立獲勝條件
* 以資訊交換、破解為主要玩法 (排除Bang!、三國殺、復仇女神號等)

# 本文列為討論的對象
* 阿瓦隆
* 亞瑟傳奇
* 狼人殺
* 驚爆倫敦

# 陣營遊戲設計方式異同
## 資訊來源
* 資訊渠道設立目的：讓玩家有機會摸清陣營，做出有利決策
* 場上共同知曉且確定正確的資訊
    * 阿瓦隆：任務結果
    * 亞瑟傳奇：任務結果
    * 狼人殺：強神、白狼王等角色的翻牌
    * 驚爆倫敦：剩下的炸彈數量
* 場上共同知曉但不確定正確性的資訊
    * 阿瓦隆：任務盾牌派票、黑白球投票
    * 亞瑟傳奇：任務盾牌派票、領袖交接順序
    * 其他玩家報出的資訊
* 僅個人知道且保證正確的資訊
    * 阿瓦隆：湖中結果
    * 亞瑟傳奇：護身符結果
    * 狼人殺：神職自己獲得的資訊（預言家、獵魔人、機械狼、惡靈板子的女巫...）
    * 驚爆倫敦：無
* 其他資訊渠道
    * 個人行為、表情...
    * 個人言論
    * 場外
## 資訊交流
* 發言（分為共同發言制和絕對發言制）
* 投票
* 其他方式
## 資訊可信度
* 第一優先：一定真實的資訊
* 第二優先：比對後可信度較高的資訊
* 第三優先：仰賴解釋、分析的資訊
* 最後順位：發言、抿牌、場外


# 阿瓦隆和狼人殺（以台式玩法為主）的比較
|  | 阿瓦隆 | 狼人殺 |
| -------- | -------- | -------- |
|   |  |   |
| 玩家數量 | 5~10，多6、8、10 | 6~12，多9、10、12 |
| 遊戲節奏 | 相對較快 | 相對較慢 |
| 玩家參與度 | 相對較高（不會有人出局） | 相對較低（首刀、首票） |
| 線上平台 | 不限 | 至少要有語音 |
|   |  |   |
| 給資訊的管道 | 任務派票, 黑白球投票 | 絕對發言, 狀態表情 |
| 給資訊的機會 | 平均 1~2 | 平均 2~3次 |
| 回應資訊的管道 | 後置位派票、**黑白球** | 後置位絕對發言、**投票票型** |
| 絕對可信的資訊 | 任務結果 | 翻牌 |

# 狼人殺玩家嘗試阿瓦隆
* 阿瓦隆的任務派票跟狼人殺的發言一樣珍貴
    * 阿瓦隆不能亂派票，相當於狼人殺不能亂發言
* 阿瓦隆的黑白球投票很重要
    * 相當於狼人殺中的票型
    * 是表達站邊最直接的方式
* 阿瓦隆的發言好但派錯票的人是壞人
    * 相當於狼人殺中瘋狂表水但大上匪票的玩家
    * 就算是好人也會先被放置在任務外，相當於狼人殺會優先票做匪事的玩家
