# 何謂異常票
* [名詞說明與簡稱](/IRsHa_XAQkq7LrMROrzVRw)中有提到
* 異常票
    * 內黑: **隊伍內**的玩家**投反對**
    * 外白: **隊伍外**的玩家**投贊成**
* **理論上**, 自己進任務能達成遊戲目標的可能性越高
    * 不會沒意義的**贊成沒有自己**的任務組合
    * 不會沒意義的**反對有自己**的任務組合
* 反過來說
    * 異常票通常是具有意義的
    * 不論是內黑外白, 通常代表在反應對這個組合的看法

# 內黑
## 藍方
* 代表意義
    * 反對這個組合整體
    * 反對這個組合內的某幾位玩家
    * 想反給這局的尾派派票, 不想讓現在的尾派結構被洗掉 
* 反對組合的代價
    * 一但開了內黑, 後續的玩家最直觀的想法就是"這人不想跟某些人一起出任務"
    * 通常大家不會知道你反對誰, 但大家會知道選了你可能再被反對一次
    * 因此一般而言你拿到的票會變少
    * 收益是: 若後續的人認為你是好人, 他派對的機會就上升了 (不派你反對的組合)
* 反給尾派的延伸
    * 一般會看到兩紅之後, 場內的人一直打黑球
    * 這也是反給後面玩家/尾派的動作
    * 代表這些人認為後面還有好人, 就算任務組合為全好也不一定要馬上出任務
## 紅方
* 作錯誤區間跟視野給場上
    * 組合內的所有好人都可能會對你有敵意
    * 如果有騙到後續的好人, 一般會誤導他們的派票
* 反給自己的尾派隊友

# 外白
## 藍方
* 代表意義
    * 贊成這個組合整體
    * 不太可能只是贊成組合內的某幾位玩家
    * 反對目前的尾派, 想要洗掉 (改變尾家)
* 控尾派
    * 不論是當下的尾派, 下一局的尾派, 甚至是雙尾派
    * 就是反對目前的尾派組合 
    * 以 10 人局而言, 就是 50 至少一壞的意思
* 請注意小心使用
    * 一場遊戲的外白次數也不會過多, 一般是 3 次以下
    * 外白過多不具任何意義, 還只會被紅方利用
## 紅方
* 幫隊友衝出去拿下任務失敗
    * 但通常衝刺完你也會壞掉
    * 因此使用時機一般會在已經沒辦法進任務後
    * 好人已經不認為你是好人, 隊友也放棄你了 => 趕快發揮你剩下的作用吧
* 控尾派
    * 假設兩紅下, 下局的最後幾個位置都是紅方
    * 表示必須在下一局的前幾家派過, 否則必為三紅
# 高風險高報酬
* 不管是內黑外白, 都代表釋放一些視角跟資訊
    * 釋放資訊的同時, 勢必要犧牲一些東西
    * 例如: 
        * 被選進任務的可能變小
        * 被誤解資訊
        * 梅林被刺可能性變高 (對尾家有敵意)
        * 紅方被抓可能性變高 (資訊太多, 能力者)
* 對紅方而言
    * 內黑外白不是必要
    * 藍方永遠可能犯錯
* 對藍方而言
    * 除非認為"雙尾派"皆為壞人, 否則外白後任務失敗, 資訊不一定更多
    * 內黑全好, 內黑給壞人尾派, 都會讓梅林很想殺人
* 10 人局的能造成實質效果的外白, 至少要兩顆 (第一局則需要三顆)
    * 如同前述, 過度外白只是被利用
    * 還必須跟隊友以及場上資訊互相配合才行
* 因此若剛接觸這款遊戲, 沒有一定的遊戲經驗, 請不要隨意開外白
