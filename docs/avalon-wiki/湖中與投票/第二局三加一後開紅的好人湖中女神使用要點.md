# 重點觀念
* 因為自己是藍方, 代表你的視角中, 原組合必定有壞人
* 若結果是兩張任務失敗, 絕對不能湖原組合 
* 由於是一藍一紅, 湖中在藍方手中, 此時要想的是如何延續湖中在藍方手上
* 並且盡可能透過自己的湖中動作讓場上認為你 (0家) 是藍方
* 這個優點是: 當 A 認 0 家是好人, 0 湖 B 好, A 可以直接有 {A, B, 0, 場上挑 A 視角中的最好玩家} 這個組合
 
# 思考要點
* 原組合是兩壞以上 or 一壞
* 第一局任務組合內, 最好的玩家是誰
* 第一局任務組合外, 最好的玩家是誰 (特別是 1~4)

# 2-1 ~ 2-4 都沒有人拆組合
* 表示至少 6~9 是默認原組合全好或至少偏好
* 若認為原組合只有一壞, 那應該湖給組合內最好的玩家 (不一定是5家, 可以是 5 家)
* 若認為原組合不只一壞, 那應該湖給 1~4 最好的玩家, 特別是視角與原組合陣營不同的玩家

# 2-1 ~ 2-4 有人拆組合 
* 特別注意: 原組合內的人有人拆
* 表示至少 6~9 大多數人認為原組合並非全好
* 湖中幾乎不能湖給原組合 (就算你是梅林看到原組合沒拇指, 也不能往任務內湖)
* 這時 6~9 拆完後給 1~4 的哪位玩家相對重要 (表示認定 1~4 的某些人比原組合好)
* 透過 6~9 的拆車資訊結合任務結果去湖 1~4 最像好人的玩家
