# 外白的功能與解讀
## 界線
這邊討論的外白，不考慮外撒組合自己幫白的情況
## 為什麼會想派到1-5、2-5才出任務?
雜魚沒有內建資訊，想多看一點派票、不同人的觀點，以期盡可能接收正確資訊
## 為什麼好人不敢外白?
+ 雜魚:收不到資訊，不敢做事，怕白到壞人被打壞
+ 梅林:怕太亮，容易死
+ 派西:怕白到拇指，保護不了梅林
## 外白要"有意識的外白"
+ 胡亂外白:我不知道這個組合怎麼樣，但我就試試看 => 組合出去，任務失敗，你並未獲得任何資訊，對於好人沒有幫助
+ 有意識外白:對於看的懂的組合，嘗試外白確認 => 我覺得偏好的組合，表達"我希望這個組合出任務"，以驗證我盤的區間是否正確
## 外置位該怎麼解讀外白?
+ 該玩家認為此任務組合偏好
+ 該玩家認為尾家(預設是5、0)是壞人，寧願現在是的任務組合，也不要給尾家派
+ 該玩家想把尾派洗給自己
    + Ex. 6白1的組合，如果任務出發，6會成為新的尾家

# 1-1~1-5
## 好人為什麼會外白?
+ 1-1~1-5 只會派三個人，需要三顆外白(沒有內黑的情況下)才會洗掉尾派，故1-1~1-5的外白通常是表達自己的視角(認為這個組合是好人)而不會以衝過為目的
+ 因 1-1~1-5沒有派票權的人為6789，故較有機會開白，以提前表達自己的視角
+ 50不太會為組合開白，因為他們是預設的派任務者，可以直接用組合表達他們的視角，外置位可以從派票及任務結果判斷他們的好壞
## 壞人為什麼會外白?
+ 奧伯倫: 無腦外白告訴大家自己壞人(不推薦)
+ 壞人位置: 如果洗尾派成功，尾家會是壞人
## 比較多外白的情況
```json
148 2+
478 12+
369 7+
127 48+
```
+ 擾亂資訊，不知道他外白是雷還是壞還是出於其他目的
+ 外置位
    + 6家雜魚視角，在開牌前可以直線認為12478同一陣線
    + 實際上的位置:4刺0娜9莫1奧，7派3梅，5是好人
    + 這樣你看懂278外白的意義了嗎?
+ 外白的人
    + 7外白:可能是認到梅林是3，看到369接收到的資訊是50拇指，於是開白梅林組合想洗掉尾派
    + 8外白:可以考慮兩種狀況
        + 認為127都好人且5偏壞，開白洗5
        + 認為5好人，4派127是在封外白(讓外白的人剩下4一個人) 
    + 2外白:在這局中相對是干擾資訊傳遞，對於雜魚判別資訊會更混亂
# 2-1~2-5
## 好人為什麼會外白?
+ 因2-1~2-5 會派四個人，只要兩顆外白就能洗掉尾派，故2-1開始通常不會有"展現視角"的白球
+ 而是希望這個任務組合出發，且洗掉0家尾派；關於洗掉0家尾派:
    + 認為0是壞人故洗掉
    + 認為這個組合和0不同隊，先看任務結果再判定0的好壞
    + 認為0是好人，任務過了以後可以再由0家派票
## 壞人為什麼會外白?
+ 出於戰術目的
+ 5家派過且0家是好人，當有組合包含隊友時，開白以爭取任務失敗的機會
+ 5家派過且0家隊友，已知2-5必定任務失敗時，提前開白以騙取好人信任 
# 3-1~3-5
## 好人為什麼會外白?
+ 兩紅情況下，認為尾派或後置位玩家是壞人時，開白以期達成任務成功
## 壞人為什麼會外白?
+ 尾派是好人且高機率派過，要開白以贏得勝利

# 小結
+ 1-1~1-5的白球多出於表現視角，2-1開始的白球多出於戰術目的
+ 不要害怕開白，但要知道自己為什麼開白
