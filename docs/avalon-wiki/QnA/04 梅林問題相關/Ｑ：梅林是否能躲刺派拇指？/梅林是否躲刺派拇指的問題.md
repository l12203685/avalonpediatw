# 梅林是否躲刺派拇指的問題

這個問題其實會涉及到當時場上的情勢與派票，如：

2017年7月19日 Alan實況中，場上派票對好人有極大的優勢，好人幾乎都選沒拇指的人，因此梅林才擁有極大的派票空間，選擇使用極壞的票型去派拇指；

2020年04月26日 線瓦，由於拇指派了自己與另一隻拇指，且正好尾家也是拇指必掉，梅林才選擇跟票派雙拇指，做成雜魚眼中的能力者。但在這局中是正好存在能夠讀懂派票的雜魚存在，在一般練習場，若其他玩家對派票敏感度沒有那麼高時，仍不推薦此種釋放資訊方式。

其他常見的梅林派拇指的較好範例，如："為了切割派票陣營、給出更多的好人區間"、"利用壞人資訊，給出更多壞人區間"、"為了做出其他資訊，先處理其他拇指"等。而像平時常見的，"為了取得雜魚認同，當使用雜魚視野派票不能不派的拇指"、"拇指票本身過高，拔不掉"等無奈狀況，最終也都有可能造成派拇指的後果產生。

但我想，"是否派拇指"的重點應該是在，是否清楚自己派拇指的理由。如果針對派拇指的思考僅是因為”我想要躲刺”，忽略場上狀況，與自身是否有將局勢搬回來的能力，那麼你的派拇指其實完全沒有任何收益，只會加速三紅。

如上面敘述，"為了切割派票陣營、給出更多的好人區間"與"利用壞人資訊，給出更多壞人"的狀況，梅林清楚知道這麼做是為了給好人"相較多好人的區間、方便好人選出偏好的組合"，非單純躲刺。。

而像"為了做出其他資訊，先處理其他拇指"，出現於當梅林無法一次處理所有拇指時，梅林就有機率會選擇邊躲刺邊放其他拇指資訊的行動，但其主要動機為釋放其他拇指資訊，亦非單純躲刺。

梅林的玩法要永遠記得自己是個資訊源，除了不被刺殺外，同時也有帶好人邁向三藍的責任。如果你覺得梅林直接自爆梅林與拇指位置是很不妥的玩法（為三藍犧牲刺殺局），那純粹地為躲刺而派拇指（為躲刺犧牲三藍）又有什麼不同呢？
