
# 梅林派票的解讀-01
* 梅林看了前面幾家的派票，要如何釋放正確資訊使50能派過？
* 不同場次的4家梅林，各自派票所對應的拇指資訊是什麼，所傳遞出去的資訊效果又是什麼？
## 梅林的課題
* 梅林的位置在1/2/3/4，都不太一樣，詳細請參閱[角色玩法-梅林篇](https://hackmd.io/@ProAvalonConcept/Bk-ub72Dn/%2FsYVk31i9R-ea6_t5nlwC3A)
* 梅林的課題在於輪到他派票時，能不能結合前面的資訊派出合理的好人組合。當你有這個認知的時候，就會知道為什麼好人要嘗試展開遊戲分支（讓梅林好打，使梅林有派票生存空間）；壞人則要封鎖資訊或誤導資訊（讓梅林難打，當梅林走投無路的時候就要去開能力突破）
* 梅林就是要在＂給資訊的強度＂與＂被發現是能力者＂中取得一個平衡點，給資訊的強度是否有符合＂當下＂大家視角的認知，這才是梅林派票的困難處。而當不符合的時候，每個人的處理方式不太一樣，有些人會選擇繼續當雜魚，有些人則會選擇直接開能力，沒有絕對正確的做法

# 小結
* 梅林需要知道＂正常多數人會對我這麼派的解讀大概為何＂，知道這件事後，就會知道即便有些人是好，但派下去可能會對雜魚/派西傳遞錯誤資訊，因此可能就不是一個好選擇
* 對於資訊的解讀，每個人習慣都不一樣，比如在派票二中，有人會覺得147是對146的修正，但也可能是在說146全好，不過當多了派票比對後，通常結論會接近
* 梅林給出的資訊如果具備某些誤導性，雖然吃刀率會下降，但相對也會犧牲好人派對的可能性，如何抉擇需自行斟酌
* 比如派票一中的梅林因為鎖定默德在35，做出違反常態梅林的行動，缺點是犧牲了尾家好人派對的可能，且一旦249才是默德，那派西會認為356有拇指；派票四的梅林派票過於保守，幾乎沒有給出資訊，且對5家有敵意感，只是剛好5家是派西且被魔甘娜派到因此被縱容；派票五的梅林雖然想讓3去當梅林，但有可能會誤導派西認為38有壞，且也可能導致3吃錯資訊派錯3-3
* 梅林的票數本來就有限，很難做出完美的派票，對於資訊的給予必然有所取捨與犧牲，有時無可避免某些誤導

# 額外補充
* 四張票跟三張票的表達不一樣，甚至於五張票的表達也不會相同
* 比如147，梅林470很常被認為是對147的延伸；如果梅林四張票的時候選4790，那是147的延伸還是1是拇指所以跳過就見仁見智；梅林47890時更是像在說1拇指
