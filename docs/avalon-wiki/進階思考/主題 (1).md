## 自我修正的循環流程
* 雜魚的遊戲流程: 假設 => 驗證 => 修正 => 假設 => ...
    * 透過假設建立派票 
    * 透過任務結果驗證假設得到資訊, 參閱[資訊的驗證](/Xg9M7rjlTNafkfc_BRj06g)
    * 透過得到的資訊修正假設, 並重新建立新假設

## 修正的契機與表達
* 需要經歷契機(變因), 才會去修正假設
    * 任務結果 
    * 湖中女神 
    * 派票回饋
    * 發言
* 修正後的新假設可以透過以下幾種方式表達
    * 第二次派票
        * 推翻第一次的派票假設
        * 修正第一次的假設壞人區間
    * 黑白球
        * 黑球: 反對你原本認同的組合
        * 白球: 贊成你原本反對的人選
        * 特別是內黑外白
            * 1-3 你派 368, 9 派 3689 你卻打內黑
            * 1-2 你派 450, 8 派 1678 你卻開外白  
    * 發言

## 修正 vs 前後矛盾
* 前後矛盾 
    * 第一次獨立/超前資訊派票(能力派), 也可說是不合理派票; 第二次推翻第一次的能力派票
    * 大部分前後矛盾的行為會被認成壞人
    * 梅林雖然是能力者, 但不太會推翻自己的派票, 會微調但不會整體推翻
* 在"沒有任何契機"下迅速用黑白球推翻自己的派票
    * 範例
    ``` json
    478 1+
    289 1+
    350
    278 134+ 
    ```
    * 3 家在 1-3 才剛派完 50, 卻又馬上在 4 家外灑的前提下外白了一個沒有 50 的組合
    * 等同在 1-3 說"我覺得 50 兩好", 卻突然在 1-4 說"我覺得50不像全好"
* 容易被當成壞人矛盾的狀況有以下幾種情形
    * 本身確實是壞人派票但又馬上需要用黑白球去衝鋒, 且任務結果確實開紅
    * 雜魚修正過早或是第一次派票太壞導致第二次派票很尷尬 (通常會當第六人除非有變因)
    * 信錯的派西維爾
        * 修正幅度超大
        * 但也是經歷了任務結果/湖中女神/派票才會認知到自己信錯
        * 如果一場修正好幾次 (信A => 信B => 信A => 信B => ...), 通常需要覆盤檢討一下
* 讓某些人突然變好人的特殊變因
    * 任務結果後此玩家變好人
    * 湖中女神拉高此玩家身份
    * 梅林或場上公認的好人選了他
* 雜魚修正
    * 通常"第一派"不要太能力, 詳細參閱[能力者](/ZBTT9muFQXWuRn-YYKA9lA)
    * 派票假設的視角若像是吃資訊, 假設錯誤後的修正也會比較容易被認為是雜魚
    * 場上風向某區間有多數壞人: 避開這些區間, 加上你的個人視角應該會有好人偏多的區間
        * 以此假設去出任務卻有失敗 => 第二次派票就會進行修正
* 注意驗證的過程
    * 任務只能用來證偽, 而不能證實 (只是無法否定假設)
    * 詳細參閱[資訊的驗證](/Xg9M7rjlTNafkfc_BRj06g)

|  | 自我修正 | 前後矛盾 |
| -------- | -------- | -------- |
| 變因  | 有   | 無  
|派票|吃資訊 |能力派
