# 範例一
* 1 家 168, 2 家思考要點會有幾種方向
    * 先當自己是忠臣思考
    * if 1 派梅閃拇指 => 79/34 有壞
    * if 1 壞人疊隊友 => 68有壞
    * if 1 壞人閃隊友 => 79/34 有壞
    * 1 不派 5 的可能原因: 因為 5 有尾派, 想給予 5 更多資訊
    * 1 不派 0 的可能原因: 因為 0 有湖中, 而且是 2-5, 先不派
## 認 1 梅林 
* 79幾乎不能派, 34 應該會有最後一隻
    * 如果認 50 兩好
        * 第二層思考: 要不要去展"50兩好"的視角 
        * 如果要展開分支我會選擇外灑 1/6/8 + 5/0 + 3/4 各選一個 
        * 或是 50 + 6/8 選一個
    * 如果認 50 有壞 
        * 50 應最多一壞 => 否則 1 會外灑
        * 第二層思考: 外撒不派50, 79有壞
        * 所以就是 1 + 6/8 + 3/4 要選一個
    * 局勢從這裡產生可能的未來分支, 詳見[增加遊戲分支](/uaUE9FZ9T-y_NMFVwURUyQ)
    * 能夠正確分3/4, 沒選到拇指, 你就會是梅林人選
## 認 1 派西 
* 79 可能有他拇指, 另一根拇指可能在 34 
    * 我自己覺得派西如果拇指在 50 不太會派168, 但沒有一定
    * 68 沒有他拇指, 但可能還是有壞 
    * 那這時候選擇是, 要不要派 79 => ... 往下拓展
## 認 1 壞人
* 可能一: 1 是單純會根據視角去派票的壞人
    * 不管是疊隊友/閃隊友 6789 幾乎都不能派
    * 所以可以乾脆的灑 5 + 340 選二 
* 可能二: 1 是會控制/創造視角的壞人
    * 隊友在 34 + 5/0, 6789 都沒有隊友 
    * 故意派 168 創造錯誤的/假視角
* 可能三: 1 是奧所以沒有任何視角
## 重點小結
* 這裡的重點不在於派了什麼組合
* 而是每種組合派票都是根據不同視角延伸出來的 
* 其他人會進而對這些派票作思考
    * 2 家的派票是否有參考 168 並展示定義
    * 是否能從 168 + 2家的派票做進一步拓展
    * 場上對 1家/ 2家的派票是否有陣營/好壞/偏能力偏雜的解讀
* 特別注意
    * 不論派 345/450/340 或是 560/580/145 都有合理解釋
    * 但一般不會選擇派 279 這種直接跟 1 完全反向派票
    * 通常這樣選完, 如果真的是 12 兩好, 會導致 12 雙下
