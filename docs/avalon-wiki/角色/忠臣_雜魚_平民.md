# 基本思考
## 有助於好人方的行為
* 好人一定要互相分享資訊
* 就算你的資訊可能是盤錯的也沒有關係
* 多練習推區間多交流資訊絕對是有利的
* 不會有好人故意做出對自己陣營不利的行為 (`表壞`)
* 在思考任務組合與投票時, 也會盡可能讓自己在別人眼中"像好人"
* 使共同區間變清晰
* 找共同的好人出任務

## 壞人在做的事情
* 封鎖資訊
* 給出錯誤資訊

# 遊戲心態
* 雜魚都想秀
* 莫甘娜都想演梅林
* 梅林只想裝雜魚
* 除了梅林的好人都想擋刀

# 不要無腦疊票
* 無腦疊票是一個很不好的習慣
    * 我們常說票多的通常都是好人
    * 但假設票多的那家是拇指那會導致梅林很難處理

# 分辨好壞能力者
* 派票的意義
    * 每位玩家派完票都代表了他假設四壞在那裏
* 運用相對好、相對壞的觀念
    * 此能力是對你相對好的還是相對壞的
    * 當然也有可能，你覺得他是壞人能力者，而他只是個雜魚去假設你是壞人而已。

# 牌局範例
## 忽略資訊傳遞的修正
* 0 家忠臣好人視角覆盤
``` json
148 4-
270
230
234
590
oox
2670
2370
1480
3790
1480
ooox
0>5 x
```
* 1-1: 148 4-
    * 4 家認為 18 有一壞
    * 1-1 就有獨立資訊偏向能力者
* 1-2: 270 
    * 完全不參考 148 的隊伍選擇
    * 因為 4 家黑球所以可以理解為重新構成隊伍組成
* 1-3: 230 
    * 認同 2 家的派票
    * 第三張票也派了 2 家所選擇的 0 家
    * 高風險高報酬的派票
* 1-4: 234
    * 認同 23 家的的資訊
    * 即 2370 都偏好
    * 18 有壞人
* 1-5: 590 oox 
    * 完全沒有參考 1~4 家的派票資訊
    * 18 可能有壞人
    * 4 認為 2370偏好
* 此時 0 家視角為
    * 59 最少一壞, 1234678 最多三壞
* 2-1, 2-2: 2670, 2370
    * 認同 2370 多好的區間
* 2-3: 1480
    * 不認同 2370 多好的區間
    * 忽視 4 的 18 一壞的內黑
    * 強行照 1 家派票
    * 有獨立資訊
* 2-4: 3790
    * 鐵咬 5 壞
    * 並認同2370多好的區間
* 2-5: 1480 ooox, 0>5 x
    * 完全忽略 234679 家的派票資訊
    * 並在派出任務失敗後湖個最少一壞沒有人撿的 5
    * 結果就是場上完全不知道 1480誰出 5本來就幾乎是壞人
    * 遊戲自然就三紅結束
* 最終結果
    * 三紅
    * 刺娜德奧派梅 849572
