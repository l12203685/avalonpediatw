# 紅方壞人階段
## 前期
* 裝無視角的角色
* 派票時嘗試模糊或誘導錯誤資訊影響雜魚
* 騙票並讓雜魚排水 (讓忠臣做梅林不會做的事)

## 中期
* 看風向切割隊友或站邊給予誤導資訊
* 站邊: 若雜魚猶疑不定, 刻意強勢衝鋒
    * 舉例: 兩紅了，四家壞人，隊友都說要在四過，若雜魚在動搖，那就不用演好人了
    * 就一起跳起來說五家偏壞四家過吧!
    * 你可以想成是壞人all in要搶第三紅
## 後期
* 或是有人提早被打入鐵壞、
* 孤立的壞人怎要看情況表壞，
* 鐵壞的壞人沒必要做好事，
* 抓住所剩不多的價值（毀湖、開白）

# 遊戲進程
* 1-4以前: 自然就好，＂刻意＂就顯得你是能力者
    * 大家都說他是好人，沒必要跳出來說他壞人
    * 大家說他是壞人，沒必要特別保住他
* 1-5當下
    * 1-5常遇到的問題是隊友票超多，我要不要派隊友，派了怕一起兩張很蠢，不派怕被發現有視野
    * 這個可以嘗試分辨他的身份、或是他的人設（他習慣不出、無腦直出、按照位階出）或是你們自己有默契
    * 以"我"(皓子) 來說，跟我常玩的幾個玩家會知道當“我尾派”的時候我派隊友，隊友「不要出」讓我決定，（僅限於還沒有出車過也就是還沒有有出者恆出、藏著恆藏的階段）但這是我跟幾個人的默契
    * 大部人依然還是用位階法: 刺客優先出，三壞同車刺客決定
    * 當然還有順位法（其實順位法不太會有兩張的問題，不過台灣大部分都打位階法）
