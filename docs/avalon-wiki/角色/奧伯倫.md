# 角色定位
* 增加遊戲的變化性
    * 拉低紅方下限
    * 提高紅方上限
    * 很容易有個人風格
* 奧伯倫大局觀
    * 玩雜魚時的思考能力
        * 奧伯倫玩的好不好就看你玩雜魚的技術好不好
        * 平常雜魚怎麼找好人跟壞人的，奧伯倫也一樣
    * 反過來盤就好: 
        * 對你沒敵意的能力者大概率是隊友
        * 有敵意的大概率是好人能力者

# 基本策略
## 任務為主
* 被選進任務就是先出失敗
* 初學者先不用考慮是否有隊友
    * 不用怕被罵，就算兩張失敗也不應該被罵
    * 統計數據顯示第一局結果兩張任務失敗 (oxx) 相比於任務成功 (ooo) 更有助於紅方拿下三紅
    * 更不要說第二局若有包到湖中女神出現兩張失敗, 還可以減低湖中女神的可信度
* 切記這只是大方向，並非定律
## 控場為輔 
* 一般來說，奧伯倫能比其他忠臣更快的找到其他紅方
    * 依照場上風向做出對紅方陣營有利的選擇
    * 如果判斷有紅方隊友可以嘗試出任務成功，讓隊友出失敗
* 要分得出來每種結果的效益並進行選擇
    * 一張失敗賺
    * 兩張失敗打平 (你還是拿下一局任務了)
    * 開藍大虧 
* 相對的如果判斷錯誤，也要有隨機應變的作法
## 喊炸車?
* 沒必要不需要做這件事
* 代表放棄每一次的練習　
* 同樣切記這只是大方向，並非定律

# 奧伯倫要點
## 前期
* 如同所有能力者, 盡量以雜魚視角思考
* 裝雜魚派票, 不讓人發現是能力者
## 後期
* 後期指的是身分暴露　(1-5 oxx, 到2-4全場沒票登出遊戲, 遊戲兩紅壞人聽牌, ... 等)
* 請找機會幫隊友衝白球
* 或是開擾亂白球 (你知道開這顆白球不會出任務, 但會讓大家難以判斷其他人陣營身分)
## 尋找紅方隊友
* 疊隊友的票
* 帶相認壞進任務
* 咬掉非隊友的好人
## 尋找好人能力者
* 從好人能力者視角找到你的隊友
* 觀察是否有人對你或某些人釋放敵意
## 欺騙藍方忠臣
* 因為不相認，不會發生隊友派票閃你的情況發生
* 盡可能讓其他人去派到你，藉此縮減梅林人選
* 不要讓其他人發現你是能力者, 越像雜魚越好
* 特別注意: 
    * 欺騙的目的就是為了最後關鍵時刻反咬一口
    * 該表壞的時候記得果斷表壞, 不要扭捏
    * 裝好人到 5-5, 梅林會感謝你, 刺客會想捅爆你
## 誤導場上資訊
* 製造錯誤區間
* 沒票的好人就盡量讓他沒票(越多好人被當壞人，你就會相對變好人出任務)
## 尾派時
* 判斷自己的身分有沒有暴露並決定行動
    * 是否要派你認為的隊友
    * 是否要派場上公認偏好的人
    * 是否要表壞 (擾亂資訊 + 幫助隊友拿下後續任務)
* 派隊友的情況
    * 目標: 一張任務失敗, 並增加任務外好人派全好的難度
        * 通常會選任務內的一個人
        * 因為你帶了隊友 => 就算你不被選, 隊友被選到的機會也大幅增加
    * 若要帶隊友不要再帶下局尾派 
        * 以 1-5 而言就是帶 0 家 + 隊友出任務
        * 通常這時候 0 家不會派隊伍內的人
        * 自然有兩個紅方的人被排除在外
    * 奧伯還是相認紅方出失敗比的是戰術與默契, 分成以下幾種情況
        * 一般情況通常是奧伯藏，讓相認出失敗
        * 奧派的很突兀, 就是來出任務失敗 => 相認紅方自己要躲好
        * 奧伯找隊友，相認壞自己出
        * 奧伯找隊友，相認也有找到，結果都不出
    * 不要派一個很好的A好人跟一個很壞的B隊友
        * 這樣很容易自然去選A = 兩壞丟外面
* 帶兩好的情況
    * 目標: 拿下任務失敗, 並拖兩個好人下水
        * 用雜魚視角派票, 讓大家無法區分第一局的壞人是誰
        * 最好是帶一個偏好跟一個模糊的好人雙下
    * 若是場上有兩個偏好的人
        * 可以全都帶上 (記得, 如果你是雜魚, 自然會帶偏好的玩家)
        * 強迫 0 家在 2-5 只能選一個偏好的人
        * 或是先讓任務成功, 強迫 0 家加進來, 再出失敗
* 找好人能力者
    * 帶上好人能力者出任務失敗，但不能做的太明顯
    * 如果好人能力者對自己沒有敵意，直接派他出失敗
    * 從可能的梅林視角確認隊友位置，隨機應變

# 結論
* 擴大隊友的優勢
* 尾派思考如何破壞任務同時混淆場上資訊
    * 讓 0 家再派紅車
    * 讓後面的好人派錯
    * 縮小其他梅林人選
    * 掩護隊友
* 如果沒有把握，那就是出失敗搶三紅
* 認壞有好的認壞做法, 最爛的做法是用嘴巴認壞
* 每個人玩法不一樣，但請記住你終究是紅方不要演好人過頭
