# 基本觀念
* 先記得以下原則
    * 要先讓別人認得下你是好人，你的資訊才傳播的出去
    * 先開三藍才有機會被刀，但是三藍被刀一樣算輸
    * 協助好人彼此連線

# 如何派票
* 前置位梅林 [1, 2, 3, 4] 的派票: 三張票
    * 第一層: 最直觀含義 這些都沒拇指
    * 第二層: 定義區間
    * 第三層: 透過前面的派票, 結合雜魚視角與旁觀視角給出好人的組合同時釋放壞人區間資訊
* 後置位梅林 [6, 7, 8, 9] 的派票: 四張票
    * 第一層: 最直觀含義 這些都沒拇指
    * 第二層: 回應前置位的派票 (鞏固派全好區間，修正派錯的) 
    * 第三層: 他會希望找誰當擋刀牌
* 大眾習慣"通常"
    * 梅林在1-1到1-4派票習慣給區間
    * 2-1開始都是給好人為主 並帶有修正意涵

## 派票基本策略
* 1-1: 給壞人的區間 [視角與區間派票法](/2h76M5FXQQelEPnlXxbyUw)
* 1-2~1-4: 依照前面的派票派出相對應的合理好人組合
* 2-1~2-4
    * 依照前面資訊派出合理的好人
    * 修正前面的錯誤區間 (修正幅度越大，越容易死)
    * 鞏固派對好人的組合
* 1-5 & 2-5: 有沒有依照前面資訊派出合理的好人組合，越不合理越容易死

# 派票區間解讀範例
* 1家派 149, 2 家派 268, 3家派 369, 4家梅林派 148
* 第一類: 梅林派票是給好人, 給你 148 就真的只是 148 都好人而已
* 第二類: 告訴你 23開拇指 56開拇指 90開拇指
* 第三類: 兩種解讀法
    * 1家派的很好 149 都沒拇指，所以我給1 讓派西知道1派很好
    * 1家派的 9 是拇指, 所以我修正148 並告訴你 23 開拇指
* 第四類: 跟前面的派票差異
    * 3家派369 4家梅林直接148 <font color=#FF0000>**完全不參考**</font>3家的369派票
    * 可以猜測梅林極度討厭369組合
    * 推論出 369 裡面應該有梅林看得到的拇指
    * 詳細思考: [梅林與雜魚視角的同步](/rq2_NSHLQ36bMndkYV19bg)
