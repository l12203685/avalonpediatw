# 大原則
* 進任務盡量出失敗
    * 大部分的遊戲, 搶先拿下任務失敗會有優勢
    * 給予梅林壓力的同時, 也可完成三紅遊戲目標
* 先搶三紅, 然後刺殺
    * 刺殺梅林可能指認錯誤
    * 三紅是直接獲勝
    * 通常通往三紅的過程會逼迫能力者釋放資訊
    * 會成為刺殺梅林的線索之一
    
## 位階法
* 從角色特性出發
    * 莫德雷德不會被梅林看到
    * 莫甘娜會被派西看到, 有機會欺騙
    * 刺客是最純粹的壞人
* 因此當相認壞同時出任務時
   * <font color=#FF0000>**三相認**</font>壞進任務, 交由刺客出失敗為最高原則
   * 位階上: 刺客 < 莫甘娜 < 莫德雷德
    * 出失敗優先順序: 刺客 > 莫甘娜 > 莫德雷德
    * 如果莫甘娜 & 莫德雷德同時進任務, 無法判斷對方分別為哪隻角色
        * 通常會先開任務成功, 彼此確認位階了 => 下回任務交由莫甘娜出失敗
    
## 出者恆出, 藏者恆藏 & 後進者出
* 若原本就有相認紅方在任務中
* 第二局任務加入了另一個相認紅方
* 這時分兩種情況
    * 第一局相認紅方已出過失敗: 由於隊友已經出過失敗, 就交由隊友繼續出任務失敗
    * 第一局相認紅方未出過失敗: 後進的人負責出失敗
* 邏輯可以想成
    * 只要出過失敗的人, 位階直接降到最低 (所以後面進任務的隊友位階必高於他)
    * 只要沒出失敗的人, 位階直接升到最高 (所以後面進任務的隊友位階必低於他)
* 延伸情況考慮
    * 莫甘娜、奧伯倫同時進第一局，且第一局是奧伯倫出失敗、莫甘娜出成功
    * 莫甘娜、刺客同時進第二局時，誰應該出失敗?
    * 任務外壞人 (刺客、莫德雷德) 視角: 
        * 不知道是失敗是奧伯倫出的 => 會認為那張失敗是莫甘娜出的
        * 因此遵從"出者恆出", 由莫甘娜"**繼續**" (任務外壞人視角) 出失敗
## 奧伯倫
* 由於奧伯倫本身就不相認且玩法多變, 因此不適合加入位階法討論
* 一樣是以大原則為優先
* 原因
    * 若出任務成功
        * 開藍 => 少拿一局任務 => 大虧
        * 開紅 => 有隊友在裡面 => 大賺
    * 若出任務失敗
        * 一張失敗 => 拿下一局任務 => 大賺
        * 兩張失敗 => 拿下一局任務但可能未來登出 => 小賺
* 假設真的兩張失敗的簡單應對法
    * 無限開白讓隊友利用你的白球打速攻衝刺隊友
