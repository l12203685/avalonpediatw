# 派西維爾角色任務
## 第二難的角色
* 因為你要辨認誰是梅林、莫甘娜
* 要讀懂梅林的派票意涵
    * 梅林會鞏固派對的區間
    * 如果梅林另外創造新的組合幾乎代表前面的人派錯了 = 他開能力 = 獨立內建資訊
* 要幫忙找莫德
* 梅林不敢咬的人你要咬
* 幫忙導湖引導風向
* 保護梅林不賣他
## 不要跳派西
* 為什麼不建議跳派
    * 壓力夠大了你還要跳派當焦點容易被找到資訊來源
    * 派西一定有最基本的內建資訊: 梅林 & 莫甘娜這兩個人一好一壞
    * 只要知道派西, 就會有從派西的視角找出誰是他的內建資訊的風險
## 派西維爾要點
* 你的雙拇指（梅娜）以外的能力者都是壞人
    * 你要判別的是派票的人是能力者還是吃資訊的雜魚
    * 能力者全部丟外面（不要選）
    * 搞事的雜魚就讓他當第六好在外面幫白
* 在前置位（1~5）還沒分出梅林時
    * 盡量不要踩拇指（梅娜）
    * 但也不要閃的太刻意 會被直接認出是派西
    * 踩錯拇指很有可能會讓後面風向直接歪掉
* 在後置位（6~0）分出梅林時
    * 可以選梅林也可以不選
    * 最重要的是不要賣梅林
* 兩紅之後不要有不能鐵咬壞人或不能選拇指（梅林）的迷思
    * 有時候莫德區間就是分不出來
    * 或是有雜魚太歪救不回來
    * 必要的時候就勇敢的鐵咬下去
    * 鐵咬魔甘娜與其他壞人都可以
    * 不鐵咬很容易派錯 或是兩好與一壞一起下去
* 同時用雜魚視角去判斷其他人派票的好壞
    * 誰偏好誰偏壞 不是只有梅林派過的才是好人
    * 你自己也要找到場上其他的鐵好與鐵壞
* 如果梅林給你的壞人與你找的壞人衝突時
    * 重新想一下 自己是不是信錯人
    * 梅林的資訊不應該前後反覆
* 不要只是看梅林選了誰
    * 要去想梅林為什麼去選這些人
    * 是有人派對 梅林希望他能擋刀 
    * 還是梅林把拇指區間夾出來
    * 又或者他單純選4個沒拇指
    * 從梅林的派票與自己定義的鐵好鐵壞結合去推論出場上的4~5好
* 雙拇指（梅娜）都說是壞人的人就一定是壞人
    * 但雙拇指（梅娜）說是好人的人不一定就是好人
    * 梅林選的人只是沒拇指 但是也可以是莫德
* 對於雜魚與壞人給的派票區間當參考就好
    * 前位的雜魚可能給錯 壞人也可以故意誤導
    * 他們不一定會是對的 因為它們不是好人能力者

# 牌局範例
* 牌局範例1 (跳派的反面教材)
    ```
    147
    246 2:我派西
    123
    689 234 外白 藍燈
    4689 123570開白藍燈
    0湖4 好人
    照派藍燈
    ```
    * 刺娜德奧 5731, 請問你會刀誰?
    * 全場有給出資訊的只有兩人, 分別是 2 家和 4 家
    * 但由於 1 家是奧伯倫, 2 家如果是派西, 在梅林還有沒有釋放資訊的時候就能準確地派出246
    * 並且在 4 家外灑689 再繼續開白
    * 代表 2 家的視野裡 24689 皆沒有拇指
    * 且 5 家高機率是壞人
    * 就算 4 家精準灑出 689 依然無法阻止 2 家梅林跳派西被四刀的下場
    * 遊戲依然是壞人勝利 (2梅林0派西)
* 牌局範例2
    * 8 派西 37拇指
    ```
    178
    267
    345
    460
    135
    oox
    2670
    3470
    ```
    * 若 7 梅林, 1 家幾乎要是壞人
    * 因為如果1是好人, 梅林大可以順著派 1780 之類的組合
